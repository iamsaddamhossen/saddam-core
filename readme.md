# Saddam Core

A reusable WordPress theme boilerplate for personal and client projects.

## Features
- Classic WordPress theme structure
- Custom Post Types (Portfolio)
- Custom Meta Fields and Shortcodes
- Helper functions for reusable code
- Enqueue CSS/JS properly
- Ready for CI/CD deployment

## Usage
1. Copy `saddam-core` to a new repository for your project.
2. Rename the theme in `style.css`.
3. Add project-specific templates, assets, and customizations.
4. Use shortcodes `[saddam_portfolio]` to display portfolio items.
